# 💎 Impact Intel: My GSSoC '26 Dashboard

I failed GSoC, but I'm not stopping! I built this dashboard to track my PRs and rank during **GirlScript Summer of Code 2026**. 🚀

![Dashboard Preview](https://github.com/vivekyadav-3/impact-intel/raw/main/src/assets/preview.png)

## 🔥 What's inside?
- **Live Countdown**: Tracking the seconds until GSSoC starts.
- **My Tech Stack**: Frontend, Backend, and Data skills.
- **Project Spotlight**: Live tracking for **Checkora** and my other targets.

## 🛠️ Tech
Built with **React, Vite, Tailwind CSS, and Framer Motion.**
